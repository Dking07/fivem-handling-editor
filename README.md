## DESCRIPTION

Transform your server's vehicle tuning experience with this sophisticated **Handling Editor**. Designed for developers and server admins, this script allows you to dynamically adjust vehicle physics and performance characteristics in real-time. Perfect for fine-tuning racing servers, roleplay servers, or just customizing your favorite cars on the fly with a sleek and modern UI!

## FEATURES

### **Dynamic Real-Time Editing**
- **Live Updates:** Adjust mass, traction, suspension, and gears in real-time.
- **Advanced State Tracking:** Probe-based SubHandling detection ensuring dynamic parsing regardless of the vehicle class.
- **Immediate Feedback:** Test your changes instantly without ensure the handling.

### **Extensive Parameter Support**
- Full access to standard `CHandlingData`.
- Support for SubHandling fields (e.g., `CCarHandlingData`, `CBikeHandlingData`, `CBoatHandlingData`).
- Precise numerical adjustments using modern UI sliders or manual inputs.

### **Export**
- Copy production-ready `handling.meta` XML syntax straight to your clipboard.

---

> ### **IMPORTANT WARNING**
> Due to limitations within the FiveM engine and native physics calculations, *some specific handling parameters* may not visibly affect the vehicle immediately upon change. In certain cases, you might need to save and restart the handling for complex physics calculations to be fully recalculated and applied by the game engine.

## HOW TO INSTALL

* [Download](https://portal.cfx.re/assets/granted-assets) script;
* Place the `dking_handlingeditor` folder inside your server's `resources` directory;
* Add `ensure dking_handlingeditor` to your `server.cfg`;
* Assign the required ACE permissions to your administrator groups to allow command usage (optional).

## HOW TO USE

### Getting Started
Use the configured command (default `/editor`) while in the driver's seat.
